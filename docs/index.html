Simple Run

A minimal GPS run tracker for iPhone. Start a run, watch your route draw on the map, and review your history.

Getting Started

Tap Start Run on the main screen. Allow location access when prompted — Simple Run uses your GPS to track distance and draw your route. Tap Finish Run when done. Your run is saved automatically.

Frequently Asked Questions

Why does the app need location access?
Simple Run uses GPS to calculate distance, pace, and draw your route on the map. Location data never leaves your device.

Where is my run data stored?
All run history is stored locally on your device. Simple Run does not use any servers or cloud storage.

The map isn't showing my location. What do I do?
Go to Settings → Privacy & Security → Location Services → Simple Run and set it to "While Using the App".
